# Workflow: Feature — V9.3

## Steps
1. Read AGENTS.md → CUTOFF.md → SKILLS-TODO.md → skills/<module>.md.
2. Check SKILLS-TODO.md for ❓ rows — fill before proceeding.
3. Apply pattern matching from ARCHITECTURE.md if needed.
4. Generate task files using generate-tasks.md.
5. Output execution plan + batch commit message.
6. Human executes per plan. Watch scope.
7. Human reviews diff → commits → pushes.
8. Log in CHANGELOG.md. Delete memory file if EPIC done.

## Rules
- No executor in task files.
- Positive instructions only in STEPS.
- Tests inline in DONE WHEN — not separate tasks.
- Execution plan is mandatory output.
